# 🎬 YouTube Card Clone (HTML, CSS, JavaScript)

A simple frontend project that replicates a YouTube video card using vanilla JavaScript.

## 🚀 Features
- Dynamic card creation using JavaScript function
- Responsive UI design using Flexbox
- Views formatting (K / M logic)
- Reusable `createCard()` function
- Clean and minimal UI

## 🧠 What I Learned
- DOM manipulation in JavaScript
- Component-like thinking in vanilla JS
- String templating using template literals
- UI structuring using HTML & CSS
- Real-world frontend design practice

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
